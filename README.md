# Masterpiece - Project Repository and Portfolio

Hello, World! This website is a personal space that brings together logical experiments, quick web utilities, and interactive projects. It serves as both a live repository of ideas and a mini personal portfolio.

---

## Features

The website is structured into independent sections:

### 1. Home and Social Links (#home)
* **Hidden Menu:** An interactive dropdown menu appears when clicking the car image.
* **Social Profiles:** Direct links to GitHub, LinkedIn, and Instagram profiles.

### 2. Binary Counter (#binary)
* **Interactive Byte Row:** You can click individual bits to flip their values between 0 and 1.
* **Real-time Readout:** Displays the corresponding decimal value (0–255) and the 8-bit binary string automatically.
* **Manual Controls:** Includes quick buttons to increment (+1), decrement (-1), and reset the counter.

### 3. Note Cards (#notes)
* **Local Storage:** Allows you to write text cards that stay saved directly in your browser.
* **Quick Sharing:** Designed with a reactive banner system to generate links and share notes immediately.

### 4. Tower of Hanoi (#hanoi)
* **Logic Puzzle:** A click-based implementation of the classic disk-stacking game.
* **Adjustable Difficulty:** A dropdown menu lets you choose between playing with 3, 4, or 5 disks.
* **Move Tracker:** Counts your moves in real time so you can track your efficiency.

### 5. Subnetting Cheat Sheet and Calculator (#subnetting)
* **Reference Guide:** Quick summaries covering octet rules, private IP ranges, and core networking formulas.
* **Prefix Table:** A reference breakdown displaying subnet masks, block sizes, and usable hosts for each CIDR prefix.
* **Built-in Calculator:** An interactive form where you type an IP address, select a CIDR prefix, and calculate network data instantly.

### 6. Roblox Games Carousel (#roblox-games)
* **Game Portfolio:** A sliding carousel navigated via arrows or dots showcasing different experiences built on Roblox.
* **Direct Links:** Each slide safely redirects players to the official game page on Roblox.
